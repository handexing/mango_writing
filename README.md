## 功能说明

- 登陆（包含第三方登录）、登出、注册、验证码
- 新建分组、修改分组、删除分组
- 新建笔记、保存、修改、删除、发布（markdown），笔记新假tag
- 根据TAG、关键字搜索笔记。
- 收藏、取消收藏笔记
- 评论、点赞
- 推荐笔记
- 笔记同步（根据差异同步数据，减少带宽消耗）
